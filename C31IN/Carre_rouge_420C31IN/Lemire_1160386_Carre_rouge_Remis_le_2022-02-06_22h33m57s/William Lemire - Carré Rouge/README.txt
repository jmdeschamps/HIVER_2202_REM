 ♦ DESCRIPTION ♦
Le but du jeu Carré Rouge est de survivre le plus longtemps possible en évitant les Sentinelles ainsi que la bordure
noire. S'il y a collision, la partie est terminée. Si vous battez le record, vous êtes désormais le nouveau champion!

------------------------------------------------------------------------------------------------------------------------

 ♦ FONCTIONNEMENT ♦
Vous devez cliquer le carré rouge et le tenir, pour ensuite tenter d'éviter une collision avec les Sentinelles, qui se
dirigent en diagonale et qui rebondissent sur les bords de la fenêtre de jeu. Leur vitesse augmente graduellement,
rendant le tout plus difficile.

Quand il y a collision, la partie est terminée et une fenêtre vous demandera votre nom et affichera votre score.

Si vous battez le record, une mention d'un nouveau record sera affichée sous le temps record, qui sera maintenant votre
nouveau record. Vous serez aussi immortalisé dans le fichier scores.txt.

Afin de réinitialiser la partie, simplement cliquer sur le carré rouge. Une fenêtre apparaît après la saisie du nom
le rappellant.

Pour afficher le record, simplement peser sur le bouton 'Meilleur Score' au bas de la fenêtre de jeu.

------------------------------------------------------------------------------------------------------------------------

♦ BUGS CONNUS ♦
La fenêtre d'alerte qui apparaît suite à la saisie du nom du joueur n'apparaît pas sur la fenêtre de jeu, mais dans le
milieu de l'écran, alors que la fenêtre de jeu est son parent.

Si vous découvrez des bugs, veuillez aviser l'auteur.
